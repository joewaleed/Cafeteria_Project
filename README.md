# Cafeteria Management System

A simple desktop application developed in **C#** for managing cafeteria data and operations using a **SQL database**. This project demonstrates basic CRUD operations, data handling, and user interface interaction for a cafeteria environment.

## 🛠️ Features

- Built with **C# (.NET Framework / .NET Core)**
- Uses **SQL scripts** (`SQLBOMB.sql`, `SQLQuery.sql`) for database schema & queries
- Designed for managing cafeteria data like orders, items, and records
- Demonstrates fundamental database interactions and Windows Forms logic

## 💡 Overview

This repository contains:
- Source code in C# for a desktop app
- SQL scripts to setup and interact with the backend database
- Images and design assets for UI/UX (if applicable)

> ⚠️ No official documentation or detailed feature design is included — extend as needed.

## 🧩 Getting Started

1. Clone the repository  
2. Open the project in **Visual Studio**
3. Import the SQL scripts into your SQL Server
4. Build and run the application

## 🧪 Technologies

- **C#**
- **SQL (T-SQL scripts)**
- **Windows Desktop UI**
